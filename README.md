# Gangeya Upadhyaya Portfolio

This repository serves as a technical portfolio for Gangeya Upadhyaya. It is a personal website built using modern web development technologies like React/Next.js and Tailwind CSS, demonstrating expertise across the entire software development lifecycle. The site's structure highlights various technical areas, including:

*   **Stack:** A comprehensive listing of technical skills (TypeScript, Node.js, Python, AWS, Azure, Docker) used for building scalable systems.
*   **Experience/Career History:** Detailed write-ups of professional roles, outlining the technologies and architectural challenges faced in large enterprise settings.
*   **Blog Section:** Articles demonstrating thought leadership on software engineering concepts and technical best practices (e.g., understanding UX, game development principles).
*   **Personal Projects:** Showcases individual implementations utilizing advanced techniques like Retrieval-Augmented Generation (RAG), local LLM inference (Ollama), cross-platform mobile architecture (Kotlin/Android), and real-time rendering (Three.js).
